# Workflow AI Sandbox

Proyecto de prueba creado exclusivamente para validar un workflow profesional de desarrollo de software asistido por inteligencia artificial.

## Objetivo

Evaluar de manera controlada:

- investigación previa;
- definición de requisitos;
- planificación;
- generación y modificación de código;
- revisión técnica;
- pruebas;
- documentación;
- control de cambios;
- reversión;
- registro de decisiones.

## Restricciones

- No contiene información real o confidencial.
- No se utilizará en producción.
- Cada cambio debe corresponder a una tarea previamente definida.
- Las herramientas de IA no podrán modificar el proyecto sin evidencia verificable.
- Los resultados serán evaluados mediante el paquete operativo del piloto.
